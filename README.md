# Travel-Template-for-SwiftUI
A Travel template app written by SwiftUI

## Installation ##
* Git and clone <br/>

```bash
$ cd /Users/user/my_project
$ git clone https://github.com/TarokhDev2020/Travel-Template-for-SwiftUI.git
```
